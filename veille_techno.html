Veille technologique

La veille technologique est une étape lors de notre formation professionnel scolaire et personnelle qui nous permet d'apprendre davantage grâce aux informations disponibles autour de nous :

  • Lieu de travail
  • Cours
  • Formation
  • Information via les recherches professionnelles : Youtube , site web etc...

L'objectif de la veille technologique est de rester informé des nouveautés, des innovations et des changements dans le secteur technologique, afin de prendre des décisions éclairées et de rester compétitif.


Différentes méthodes peuvent être appliquées pour rester informé sur l'actualité informatique, mais la meilleure selon moi est de s'abonner à des sites web qui postent des articles et aussi crée des alertes google pour avoir une notification tous les jours ou toutes les semaines.
Il est important de créer des flux RSS notamment possibles avec le site feedly.

Laravel

Laravel est un framework web open-source écrit en PHP respectant le principe modèle-vue-contrôleur et entièrement développé en programmation orientée objet.
Laravel est distribué sous licence MIT, avec ses sources hébergées sur GitHub.

Parmi les nombreux framework web open-source Laravel est l'un des plus utilisés avec codeigniter et symfony.
Mais son utilisation et sa documentation reste selon moi plus simple à comprendre pour des étudiants.
Avec Laravel on peut crée son Api WEB de façon très complete.

  • Base de données : MySQL avec connexion par utilisateurs et Mot de passe
  • Page de connexion et d'inscription web qui ajoute un utilisateur sur la base de données
  • Langage blade qui s'implifie la création des vues : utiliser un gabarit pour plusieurs vues
  • Les seeders avec fakers PHP pour crée de faux clients avec leurs informations
  • Permet de creer un CRUD outil permettant d'ajouter des articles sans aller dans les seeders
  • Permet d'integrer du react, ajax
  • Et pleins d'autres fonctionnalités

Dans le cadre de l'obtention de mon BTS pour l'epreuve E5 je realise un projet en Laravel appeler My sneakers un e-shop de sneakers (chaussures rare).


C# .net , framework WPF

C# est un langage de programmation développé par Microsoft, orienté objet et principalement utilisé pour développer des applications sur la plateforme .NET.

WPF, ou Windows Presentation Foundation, est un framework de développement d'interfaces utilisateur graphiques pour les applications Windows. Il offre une approche déclarative pour la création d'interfaces riches et interactives, en utilisant XAML pour la conception et le style des éléments graphiques.

  • Gestion de base de données : C# avec WPF permet de se connecter et d'interagir avec différentes bases de données, telles que Microsoft SQL Server, pour stocker et récupérer les données.
  • Interface utilisateur : Grâce à WPF, il est possible de créer des interfaces utilisateur .

Dans le cadre de l'obtention de mon BTS pour l'epreuve E5 je realise un projet en C# .NET en utilisant WPF appeler GestionStockMySneakers. Le projet consite a faire une application qui permet de gerer le stock des articles disponibles sur le site.
